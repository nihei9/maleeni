# Changelog

## v0.1.0

* maleeni v0.1.0, this is the first release, supports the following features.
  * Definitions of lexical specifications by regular expressions
	* Alternation: `|`
	* Dot expression: `.`
	* Bracket expressions: `[...]`, `[^...]`
	* Code point expressions: `\u{...}`
	* Character property expressions: `\p{Letter}`
	* Repetition: `*`, `+`, `?`
	* Grouping: `(...)`
	* Fragment: `\f{...}`
  * Mode transition
